## CSS

- CSS のディレクトリを `docs/css` から `dist` に変更したのでパスを修正

## SCSS

- インポートディレクトリを変更
- 依存ファイルの import 削除
- `$use-xxx` は 次世代 Sass の use と紛らわしいので `$option` に変更
  - `$use-component-xxx` 周りの変数名を変更
  - `$use-theme-xxx` 周りの変数名を変更
- Utilities：Important を付与するオプションを追加 #256
- Utilities：Overflow・Ellipsis・Scroll を統合してリネーム #261
- Global：Floating・Focus 変数の集約 #263
- Section の `.inner-vw` は廃止
- Grid の `$column-class` は `$selector-column` に変更
- Grid の `$grid-split` は `$flex-split` に変更（定義も Flex に移動）
- Grid：Gap の Row・Column をリネーム #254
- Box の `.is-bg-cover` などはコンポーネントに書いた方が良いので削除
- Box の `is-angle-(xxx)` から Padding を削除
- Card の `.card` はデフォルトで `overflow: hidden;` するように変更
- Card の flexbox は box と用途を迷うので削除
- Joint の `$joint-class` は `selector-joint` に変更
- Text の `$text-class` は `$selector-text` に変更
- Button の `$button-class` は `$selector-button` に変更
- Button：役割ごとの Floating 設定を可能にする #262
- Icon の `$icon-class` は `$selector-icon` に変更
- WYSIWYG：別のエレメントに依存していた変数を分離 #248
- Flex：メディアクエリ指定無しのモディファイアを追加 #260
- Space：Row・Column を 4 方向に分割 #255
- Margin：ネガティブマージンの追加 #257
- Padding：縦横ショートカットをリネーム #258
- Centering：Margin Auto 拡張につき廃止 #259

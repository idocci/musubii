## Detail

各スタイル用の条件分岐可能な色変数。[Theme Light](/configs/themes/theme-light)・[Theme Dark](/configs/themes/theme-dark) を読み込み、[CSS Variables](/getting-started/css-variables) の有効化や [Default Theme の変更](/getting-started/dark-mode#default-dark) を処理します。

例えば、プロジェクトのテーマカラーに `$convert-primary` を使っておけば、ダークモード対応する場合に自動で CSS Variables の変数を出力。レガシーブラウザ対応で CSS Variables 未使用の CSS を出力する場合は静的な値を出力するなど、臨機応変な対応が可能となります。

# LICENSE

## CC0 1.0 Public Domain

v7.0.0 以降、ライセンスを MIT から CC0-1.0 へ変更しました。著者情報をサイトやソースコードに記載せずに商用利用可能です。→ [詳細 - GitHub Issues](https://github.com/qrac/musubii/issues/249)

変更した主な理由は、SCSS を使った部分利用時にライセンス明記の制約が現実的でないことです。

MUSUBii は Web 制作において下地となるレイヤーのライブラリで、その実態はありふれた CSS の集合体ですが、Web コーディングの面倒な時間を確実に減らせます。

プロジェクトによっては一部しか利用しない場合もありますし、そもそも使っていることすら忘れて欲しいほど制作物に馴染んで欲しい存在です。

著者情報をどこにどう書けばいいのか・受託制作に使っても良いのか・部分的に CSS をコピーしても良いのか、今後発生しうる非生産的な確認の時間すべてを捨て去りたい。

開発者としては、著者情報の明記よりも Web 制作を行う利用者のストレスが減ることを望みます。

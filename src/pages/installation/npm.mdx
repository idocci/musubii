## Install

MUSUBii のインストールは npm または Yarn で行います。

```
npm install musubii
```

```
yarn add musubii
```

## CSS Path

CSS ファイルは `package.json` で main に設定されています。

```
musubii/dist/musubii.css
```

```
musubii/dist/musubii.min.css
```

[[warning | Warning]]
| CSS ファイルをそのまま使う場合、[CSS Variables](/getting-started/css-variables) は有効化できません。

## SCSS Path

SCSS ファイルは、コンフィグとスタイルを分けて読み込めます。

```
musubii/src/scss/configs/_import.scss
```

```
musubii/src/scss/styles/_import.scss
```

分ける必要が無ければ一括で読み込みます。

```
musubii/src/scss/musubii.scss
```

## CLI Build

npm scripts で SCSS をコンパイルする場合は `node-sass` を利用します。

```json
{
  "scripts": {
    "build": "node-sass ./src/scss -o ./dist/assets/css --include-path ./node_modules --output-style expanded"
  }
}
```

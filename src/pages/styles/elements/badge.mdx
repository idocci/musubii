## Detail

バッジ装飾を作るエレメント CSS。

## Basic

<PreviewBadgeBasic />

| Selector | Description                            |
| -------- | -------------------------------------- |
| `.badge` | バッジ装飾の起点として使うセレクター。 |

[[warning | Warning]]
| `.badge` には `.is-plain` `.is-outline` どちらかの付与が必須です。

| Modifier (One is required) | Description                                               |
| -------------------------- | --------------------------------------------------------- |
| `.is-plain`                | `.badge` に付与すると塗りベースのボタンになります。       |
| `.is-outline`              | `.badge` に付与するとアウトラインのあるボタンになります。 |

| Modifier       | Description                                   |
| -------------- | --------------------------------------------- |
| `.is-strong`   | `.badge` に付与すると太文字になります。       |
| `.is-round`    | `.badge` に付与すると角が完全に丸くなります。 |
| `.is-disabled` | `.badge` に付与すると非活性になります。       |

## Role

`.is-(role)` 形式のモディファイアを `.button` に付与すると、各役割毎のカラーリングに変化します。

| Primary       | Info       | Success       | Warning       | Danger       |
| ------------- | ---------- | ------------- | ------------- | ------------ |
| `.is-primary` | `.is-info` | `.is-success` | `.is-warning` | `.is-danger` |

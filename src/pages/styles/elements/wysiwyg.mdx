## Detail

WYSIWYG や Markdown から出力された「class 名の無いプレーンな HTML タグ」に対して、一括で装飾を施すエレメント CSS。[Text](/styles/elements/text)・[List](/styles/elements/list)・[Table](/styles/elements/table) と近い見た目を簡単に作れます。

## Basic

`.wysiwyg` をラッパーとして装飾を施したいエリアに設定します。

| Selector   | Description                              |
| ---------- | ---------------------------------------- |
| `.wysiwyg` | WYSIWYG 装飾の起点として使うセレクター。 |

## Link

`.wysiwyg` 内の `a` タグは自動的に下線リンク装飾となります。`.wysiwyg` に `.is-link-reverse` を付与するとホバーで下線が表示されるよう変更できます。

デフォルトは `border-bottom` プロパティが使われていますが、[Option](/configs/variables/option#option-add-style-xxxx) の `$option-add-style-wysiwyg-link-border` を `false` にすると、下線が `text-decoration: underline;` に切り替わります。

<PreviewWysiwygLink />

| Selector | Description                             |
| -------- | --------------------------------------- |
| `a`      | `.wysiwyg` 内で装飾されるアンカータグ。 |

| Modifier           | Description                                                             |
| ------------------ | ----------------------------------------------------------------------- |
| `.is-link-reverse` | `.wysiwyg` に付与するとアンカーがホバーで下線表示されるようになります。 |

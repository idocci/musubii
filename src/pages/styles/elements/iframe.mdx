## Basic

`.iframe-wrap` で `.iframe` を ラップすることで、レスポンシブなアスペクト比維持を行えます。

| Selector       | Description                                              |
| -------------- | -------------------------------------------------------- |
| `.iframe-wrap` | `.iframe` をラップしてアスペクト比を指定するセレクター。 |
| `.iframe`      | iframe をターゲットにするセレクター。                    |

[[warning | Warning]]
| `.iframe-wrap` には `.is-aspect-16x9` `.is-aspect-4x3` どちらかの付与が必須です。

| Modifier          | Description                                                 |
| ----------------- | ----------------------------------------------------------- |
| `.is-aspect-16x9` | `.iframe-wrap` に付与するとアスペクト比が 16:9 になります。 |
| `.is-aspect-4x3`  | `.iframe-wrap` に付与するとアスペクト比が 4:3 になります。  |

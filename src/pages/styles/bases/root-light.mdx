## Change Selector

`$selector-root-light` をオーバーライドすることで、出力を `:root` 以外の場所へ移すことができます。

```scss
// 出力をbodyに変更したい場合
$selector-root-light: "body"; // Default: ":root"

// Import
@import "~musubii/src/scss/configs/_import.scss";
```

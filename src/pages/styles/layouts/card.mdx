## Detail

カード型の汎用レイアウトを作る CSS。

## Basic

`.card` セレクターを使用します。余白を作る場合は [Padding](/styles/utilities/padding) を付与。配下には [Box](/styles/layouts/box)・その他エレメントやコンポーネントを配置します。

<PreviewCardBasic />

| Selector | Description                        |
| -------- | ---------------------------------- |
| `.card`  | カードレイアウトに使うセレクター。 |

| Modifier               | Description                                            |
| ---------------------- | ------------------------------------------------------ |
| `.is-bg, .is-bg-light` | `.card` に付与すると白背景が追加されます。             |
| `.is-outline`          | `.card` に付与すると `border` が追加されます。         |
| `.is-floating`         | `.card` に付与するとフローティング装飾が追加されます。 |

## Radius

`.is-radius-(value)` 形式のモディファイアを `.card` に付与すると溝を作れます。`(value)` を省略した場合は `md` と同等の値になります。

`(value)` は [Meter](/configs/variables/meter) の値を引き継いでいます。

<PreviewCardRadius />

| XXS              | XS              | Small           | Medium                     | Large           | XL              | XXL              |
| ---------------- | --------------- | --------------- | -------------------------- | --------------- | --------------- | ---------------- |
| `.is-radius-xxs` | `.is-radius-xs` | `.is-radius-sm` | `is-radius, .is-radius-md` | `.is-radius-lg` | `.is-radius-xl` | `.is-radius-xxl` |

## Link

<PreviewCardLink />

| Modifier       | Description                                        |
| -------------- | -------------------------------------------------- |
| `.is-link`     | `.card` に付与するとリンクホバー装飾を追加します。 |
| `.is-disabled` | `.card` に付与するとリンクを無効化します。         |

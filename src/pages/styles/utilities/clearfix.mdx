# Clearfix

## Table of Contents

contents list

## Detail

共通モディファイアで `float` プロパティを解除するユーティリティ CSS。

## Basic

`.is-clearfix` を解除したいセレクターに付与します。

| Modifier       | Description                                                 |
| -------------- | ----------------------------------------------------------- |
| `.is-clearfix` | 擬似要素 `:after` を挿入して `float` プロパティを解除する。 |

## Important

[Option](/configs/variables/option#option-important-xxxx) の `$option-important-clearfix` を `true` にすると、全プロパティに `!important` を付与できます。
